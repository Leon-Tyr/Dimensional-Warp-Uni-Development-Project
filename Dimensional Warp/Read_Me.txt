Instructions for Dimensional Warp Project -Unity-

To run the project within Unity,  Unity 2019 version 1.5f1 needs to be installed. 
After the project needs to be opened via Unity Hub but the scripts can be accessed from the files 
and can be opened with visual studios or another similar software.

To run the .APK on android device, the device must allow for installation of unknown apps and
the device version must be Android 7.0 or higher. 

To access the augmented reality feature the android device must also install 	
"Google Play Services for AR" aka "AR Core" from the Android Play Store.
